# Kemy's Navigation Tools


This mod adds several navigational tools to sailwind.

## Installation

>  This mod requires Bepinex to run : https://www.overwolf.com/app/thunderstore-thunderstore_mod_manager

Extract the .zip file from the latest release into your Bepinex>Plugins Folder.

## Features

### Inclinometer

<img width="851" height="824" alt="image" src="https://github.com/user-attachments/assets/2083ecd6-f372-4355-b27b-fd77e5178a11" />
This tool enables you to read the current inclination of your ship. The free swinging arm always faces down, so place it or attach it to a wall to measure your ship's Roll or Pitch. 
Currently available at the Navigational tool merchants in Gold Rock City, Fort Aestrin and Dragon Cliffs.

### Bearing Compass

<img width="1021" height="1105" alt="image" src="https://github.com/user-attachments/assets/eb2b51db-20b3-4aa1-9d0f-d1e2c90a38f6" />
This compass enables you to take an exact bearing to a target. It features a foldable sighting vane that lets you aim directly at the target you wish to get the bearing of. The sight will automatically fold away when you put the compass down, allowing you to still read it as a regular compass.
The compass features 2 scales, and inner and outer. The outer scale works as a normal compass bearing scale, with 1 degree accuracy and numbered degrees every 10 degrees. The inner ring is a mirrored degree ring, allowing you to more easily see the bearing on the near side.
